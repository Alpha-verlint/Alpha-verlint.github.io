小美打怪题目链接

首先分析一下题目,小美每次打败怪物都会进行复制粘贴,自己变身为怪物(佛耶戈?),并接着寻找下一个数值都在自己之下的怪物进行复制粘贴,我们要求出一条最优路径,使得小妹能够打败尽可能多的怪物.在这条最优路径上,我们可以知道怪物的两个数值都是单调的,那么我们可以通过把所有小妹能打败的怪物的集合排序,比如对第一个数据进行排序,那么最优路径上的怪物的第二个数据一定也是单调的,那么我们可以把问题转化为求所有第二个数据排列成一个无序序列,找出这个无需序列的最长子序列即是答案.

大体思路:分离出小美当前能打败的怪物的集合,并根据第一个数据进行排序,求取出第二个数据的最长子序列.

详细细节看代码:

#include <iostream>
#include <algorithm>
#include <vector>

using namespace std;

const int N = 1010;

int fa[N],fb[N],ans[N];

vector<pair<int,int>> ar;

int n,h,a;

int main()
{
    cin >> n >> h >> a;
    for (int i = 0;i < n;i ++) ans[i] = 1;
    for (int i = 0;i < n;i ++) cin >> fa[i];
    for (int i = 0;i < n;i ++) cin >> fb[i];
    for (int i = 0;i < n;i ++)
        if (h > fa[i] && a > fb[i])
            ar.push_back({fa[i],fb[i]});
    sort(ar.begin(),ar.end());//这里我们从小到大排序
    for (int i = 0;i < ar.size();i ++)
        for (int j = 0;j < i;j ++)  
            if (ar[i].first > ar[j].first && ar[i].second > ar[j].second)//因为第一个数据可能相等,我们需要对两个数据都进行判断.
                ans[i] = max(ans[i],ans[j] + 1);//每个怪物可以选择打或者不打,每次比较打或者不打哪个更优.
    int res = 0;
    for (int i = 0;i < ar.size();i ++) res = max(res,ans[i]);
    cout << res;
    return 0;
}
